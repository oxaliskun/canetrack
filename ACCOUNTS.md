# CaneTrack Default Accounts

This document lists the pre-configured accounts for the CaneTrack system. These accounts have been restored after the database reset.

## 🔐 Default Credentials
All accounts use the same password for testing convenience.

- **Password**: `password`

### 🛠️ Admin Account
- **Email**: `admin@canetrack.com`
- **Role**: `ADMIN`
- **Access**: Full system control, user management, and reporting.

### 🚛 Operator Account (Mill Gate)
- **Email**: `operator@canetrack.com`
- **Role**: `OPERATOR`
- **Access**: Create and manage weight tickets at the mill.

### 🏭 Receiver Account (Refinery)
- **Email**: `receiver@canetrack.com`
- **Role**: `RECEIVER`
- **Access**: Record refinery weights and reconcile deliveries.

### 🌾 Farmer Account (Sample)
- **Email**: `farmer@canetrack.com`
- **Role**: `FARMER`
- **Access**: View personal delivery history and earnings.

---

## 📝 Registration Policy
- **Farmers**: Can register themselves via the "Create Account" link on the login page.
- **Staff (Operators/Receivers)**: Must be created by an Admin through the **System Users** section in the Admin Dashboard.
